# CedarLicenseTool v1.0.0 发布说明


## 概述

CedarLicenseTool 是一款许可证分发工具，用于为客户生成授权许可证文件，无需登录管理后台即可完成许可证的创建与导出。

生成的许可证与 license-manager 完全兼容。


## 新增功能

### 密钥管理

- 一键生成 RSA 密钥对，自动保存到本地
- 应用启动时自动加载已有密钥，无需重复生成
- 支持复制私钥或公钥到剪贴板

### 许可证生成

- 填写客户名称和机器指纹即可生成许可证
- 灵活授权时长：支持快速选择（1/3/6个月，1/2/3年）或自定义时长
- 有期限 / 无期限两种授权模式
- 支持 JSON 格式的扩展功能配置

### 导出与分发

- 许可证以 Base64 编码展示，可直接复制
- 下载为 `.lic` 文件保存到本地
- 一键导出存档包（包含公钥和许可证文件），方便交付给客户


## 加密说明

- 签名算法：RSA-PSS + SHA-256
- 与 license-manager完全兼容
